## Description

<p>This is a simple web application that takes your birthday <br> and calculates the day of the week
you were born <br> and then depending on gender, it outputs your Akan name.</p>

### Installation Process
HTTPS: git clone https://github.com/robinadoro/Akan-Naming

## Behaviour Driven Development
User fills the birthdate
User selects gender
User submits information to reveal name
